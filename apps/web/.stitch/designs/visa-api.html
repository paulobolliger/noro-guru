Como funciona a integração

01

Obtenha sua Chave

Crie sua conta e gere uma `API_KEY` instantânea para começar a fazer requisições.

02

Configure os Parâmetros

Envie a origem e o destino para obter os requisitos detalhados de entrada.

03

Exiba para o Usuário

Utilize nossos payloads estruturados para montar sua interface de check-in ou vendas.

# Fazer uma consulta simples
curl -X GET "https://api.noro.guru/v1/visa/requirements" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -G \
  -d "origin=BR" \
  -d "destination=PT"
const response = await fetch("https://api.noro.guru/v1/...", {
