## Decisao Atual

Integracoes sociais OAuth ainda nao sao arquitetura vigente.

Antes de implementar, validar:

- dominio oficial de callback;
- app responsavel pelos endpoints;
- escopos atuais por provider;
- modelo de tokens por tenant;
- revisao de app em cada plataforma.
